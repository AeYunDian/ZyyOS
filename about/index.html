网站简介

站名：Zyy Maple Leaf OS - 官方网站

建站日期：2026-01-20

技术实现

域名:本站使用多个域名，都通过阿里云注册，并由 Cloudflare 提供 DNS 解析和 SSL 证书。

证书:多个域名均持有 中国互联网络信息中心（CNIC） 颁发的合法域名证书，注册人信息已通过实名认证。

安全加密：全站使用 HTTPS 加密访问。

托管:网站源码托管于 GitHub，并通过 GitHub Pages 自动部署。

关于作品

这个作品只是一个用Scratch制作的伪操作系统，并不是真正的硬件操作系统，因错误理理解而导致的任何后果，我们将不承担！
