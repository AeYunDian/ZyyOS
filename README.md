## ✨ 项目简介

Zyy Maple Leaf OS 是一款基于 Scratch 平台开发的伪操作系统。它打破了传统 Scratch 项目的局限，通过精心设计的交互动画和强大的 UI 界面，为用户带来了极致的沉浸式编程体验。无论你是 Scratch 爱好者还是编程初学者，都能在这里找到属于你的乐趣。

## 🌟 核心特性

- **🎨 精美 UI 设计**：采用枫叶（Maple Leaf）主题配色，界面清新优雅，视觉体验极佳。
- **🎬 丝滑流畅动画**：内置大量精心调优的过渡动画与交互特效，操作手感流畅自然。
- **💻 强大的伪 OS 功能**：模拟真实操作系统的文件管理、多窗口切换、进程管理等功能。
- **🧩 极致的编程体验**：底层逻辑严谨，代码结构清晰，为 Scratch 高阶编程提供了优秀的范本。

## 🚀 快速开始

### 环境要求
- 推荐使用Windows访问
- 不需要支持 Scratch 3.0 运行环境的平台

### 如何运行
1. **在线体验**：直接访问我们的项目主页 [https://zyyos.undz.cn/download/]。
2. **本地运行**：
   - 下载文件（.exe 格式）。
   - 注：不需要打开 Scratch 哦~。
   - 打开文件即可开始体验！

## 📂 项目目录/架构说明

- **桌面与任务栏**：负责全局导航与多任务管理。
- **设置面板**：支持个性化主题切换与系统参数调整。
